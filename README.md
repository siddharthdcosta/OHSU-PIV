# Particle-Image-Velocimetry-3D

contains analysis code to quantify **mechanical outputs**—**shear strain**, **shear stress**, and **traction forces**—from **live-cell imaging** of **glaucomatous trabecular meshwork (TM) cells** cultured on / embedded in a **PIV (particle image velocimetry) gel**. The core workflow takes time-lapse microscopy image sequences of embedded fluorescent beads (or other fiducials), estimates **3D displacement/velocity fields** via PIV, then converts those fields into physically meaningful biomechanical maps
